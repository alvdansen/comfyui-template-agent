**src/document/models.py** -- Pydantic models:
- `IOItem(BaseModel)`: nodeId (int), nodeType (str), fieldName (str, default ""), mediaType (str, default ""). Represents one input or output node.
- `IOSpec(BaseModel)`: inputs (list[IOItem]), outputs (list[IOItem]).
- `IndexEntry(BaseModel)`: All fields matching the Template model in src/templates/models.py: name (str), title (str), description (str, default ""), mediaType (str, default ""), mediaSubtype (str, default ""), tags (list[str], default []), models (list[str], default []), requiresCustomNodes (list[str], default []), date (str, default ""), openSource (bool, default True), size (int, default 0), vram (int, default 0), usage (int, default 0), searchRank (int, default 0), username (str, default ""), io (IOSpec, default_factory), thumbnail (list[str], default []).
- `NotionSubmission(BaseModel)`: template_name (str), title (str), description (str), models (list[str]), custom_nodes (list[str]), io_summary (str), thumbnail_notes (str), workflow_link (str, default "").

**src/document/metadata.py** -- Core logic:
- `_load_core_nodes() -> set[str]`: Load data/core_nodes.json using `Path(__file__).parent.parent.parent / "data" / "core_nodes.json"`. Return as set.
- `extract_io_spec(workflow: dict) -> IOSpec`: Parse workflow nodes list. INPUT nodes: LoadImage, LoadVideo, VHS_LoadVideo, LoadAudio -- create IOItem with nodeId, nodeType, mediaType inferred from type name (LoadImage -> "image", LoadVideo/VHS_LoadVideo -> "video", LoadAudio -> "audio"). OUTPUT nodes: SaveImage, PreviewImage, SaveAnimatedWEBP, VHS_VideoCombine, SaveAudio -- mediaType inferred similarly ("image", "video", "audio"). Also recurse into definitions.subgraphs[].nodes for subgraph-wrapped IO. Return IOSpec with inputs and outputs lists.
- `_detect_models(workflow: dict) -> list[str]`: Find model loader nodes by type name containing "Loader" or "Load" AND having a widgets_values list. Extract the first string element from widgets_values that looks like a model path (contains "/" or ends with ".safetensors"/".ckpt"/".pt"/".pth"/".bin"). Return sorted unique list. Common loader types: CheckpointLoaderSimple, UNETLoader, CLIPLoader, LoraLoader, ControlNetLoader, VAELoader.
- `_detect_media_type(workflow: dict) -> str`: Check output node types. If any SaveImage/PreviewImage -> "image". If any VHS_VideoCombine/SaveAnimatedWEBP -> "video". If any SaveAudio -> "audio". Default "image".
- `_detect_custom_nodes(workflow: dict) -> list[str]`: Get all node types via extract_node_types (import from src.templates.fetch). Load core nodes via _load_core_nodes(). Return sorted list of node types NOT in core nodes set, filtering out UUID-pattern types (subgraph refs).
- `generate_index_entry(workflow: dict, name: str, title: str = "", description: str = "", tags: list[str] | None = None, username: str = "", vram: int = 0, size: int = 0) -> IndexEntry`: Format-gate with detect_format (raise ValueError if not "workflow"). Call extract_io_spec, _detect_models, _detect_media_type, _detect_custom_nodes. Build IndexEntry with auto-detected fields + user-provided overrides. Set date to today (datetime.date.today().isoformat()). If title empty, generate from name (replace hyphens with spaces, title case). Return IndexEntry.
- `format_index_entry(entry: IndexEntry) -> str`: Return entry.model_dump() as pretty-printed JSON string (json.dumps with indent=2).

**src/document/notion.py** -- Notion markdown and thumbnail reminders:
- `_load_thumbnail_specs() -> dict`: Load data/guidelines.json, find rule with id "thumbnail_specs", return its dict. Use same path pattern as metadata.py.
- `thumbnail_reminder() -> str`: Load thumbnail specs. Return formatted string with: "## Thumbnail Requirements\n\n" then bullet points: "- Ratio: 1:1", "- Video thumbnails: 3-5 seconds", "- Use workflow output (effect preview), NOT screenshots", "- Keep style consistent with existing templates", "- Avoid key info in top-left corner (API badge goes there)", "- Supported types: https://github.com/Comfy-Org/workflow_templates?tab=readme-ov-file#4--choose-thumbnail-type".
- `generate_notion_markdown(entry: IndexEntry, workflow_link: str = "", extra_notes: str = "") -> str`: Generate copy-pasteable Notion submission markdown. Format:
  ```
  # Template Submission: {entry.title}

  ## Workflow Details
  - **Name:** {entry.name}
  - **Media Type:** {entry.mediaType}
  - **Date:** {entry.date}
  - **Author:** {entry.username}
  {if workflow_link: "- **Workflow Link:** {workflow_link}"}

  ## Models Required
  {bullet list of entry.models, or "None (core nodes only)"}

  ## Custom Node Dependencies
  {bullet list of entry.requiresCustomNodes, or "None"}

  ## Inputs & Outputs
  **Inputs:** {comma-separated list of io input nodeTypes, or "None"}
  **Outputs:** {comma-separated list of io output nodeTypes, or "None"}

  ## Tags
  {comma-separated entry.tags, or "None specified"}

  ## Description
  {entry.description or "[Add description here]"}

  {if extra_notes: "## Notes\n{extra_notes}"}

  ---
  {thumbnail_reminder()}
  ```
- `format_notion_markdown(markdown: str) -> str`: Identity function (returns as-is). Exists for pattern consistency with other modules' format_* functions.
