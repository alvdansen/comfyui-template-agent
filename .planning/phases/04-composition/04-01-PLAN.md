Create `src/composer/graph.py` with `WorkflowGraph` class:

```python
class WorkflowGraph:
    """Type-safe ComfyUI workflow graph builder."""
```

Internal state:
- `_nodes: dict[int, GraphNode]` — node ID to node
- `_links: list[GraphLink]` — all connections
- `_specs: NodeSpecCache` — injected or created empty
- `_next_node_id: int = 1`
- `_next_link_id: int = 1`
- `_groups: list[dict] = []`
- `_definitions: dict = {"subgraphs": []}`
- `_extra: dict = {}`

Constructor: `__init__(self, specs: NodeSpecCache | None = None)` — uses provided cache or creates empty one.

Methods:

`add_node(node_type: str, title: str = "", pos: list[float] | None = None, widgets_values: list | None = None) -> int`:
- Look up spec from `_specs` (if available).
- If spec exists and widgets_values is None, auto-populate widgets_values from spec defaults. Iterate spec.inputs_required then spec.inputs_optional in order, include only inputs where is_widget=True. Use each input's default value (or "" for STRING, 0 for INT/FLOAT, False for BOOLEAN, first option for COMBO).
- Create GraphNode with: auto-incremented id, type=node_type, pos (default [0,0]), size=[315,170], flags={}, order=0, mode=0, properties={"Node name for S&R": node_type}, widgets_values, title, inputs=[] (populated from spec connection inputs), outputs=[] (populated from spec outputs).
- For each connection-type input in spec (is_widget=False), add to node.inputs: {"name": input_name, "type": input_type, "link": None}.
- For each output in spec, add to node.outputs: {"name": output.name, "type": output.type, "links": []}.
- Store node, increment counter, return node ID.

`connect(src_node_id: int, src_output: int | str, tgt_node_id: int, tgt_input: int | str) -> int`:
- Resolve slot indices: if str, find by name in node's outputs/inputs lists.
- Get output type from source node's outputs[slot].type.
- Get input type from target node's inputs[slot].type.
- Call check_type_compatibility(output_type, input_type). Raise TypeError if incompatible with message: f"Cannot connect {output_type} to {input_type}".
- Wildcard "*" matches anything.
- Create link: GraphLink(link_id=next, origin_node_id=src, origin_slot=slot, target_node_id=tgt, target_slot=slot, type=output_type).
- Update source node outputs[slot].links to include link_id.
- Update target node inputs[slot].link to link_id.
- Return link_id.

`set_widget(node_id: int, widget_name: str, value) -> None`:
- Get node. Get spec from _specs (if available).
- Find widget position: iterate spec required+optional inputs in order, counting only is_widget=True inputs. When name matches widget_name, that's the index.
- If spec has COMBO options for this widget, validate value is in the list. Raise ValueError if not.
- If spec has min/max, validate numeric range.
- Set node.widgets_values[index] = value.

`remove_node(node_id: int) -> None`:
- Remove all links where origin_node_id or target_node_id equals node_id.
- Clean up link references in remaining nodes' inputs/outputs.
- Delete node from _nodes.

`serialize() -> dict`:
- Build workflow dict:
  - "last_node_id": max(node IDs) or 0
  - "last_link_id": max(link IDs) or 0
  - "nodes": list of node dicts (from GraphNode model_dump, excluding empty optional fields)
  - "links": list of link arrays [link_id, origin_id, origin_slot, target_id, target_slot, type]
  - "groups": self._groups
  - "config": {}
  - "extra": self._extra
  - "version": 0.4
- If self._definitions["subgraphs"]: include "definitions".
- Assert detect_format(result) == "workflow" as a safety gate.
- Return dict.

Add a module-level helper:
`check_type_compatibility(output_type: str, input_type: str) -> bool` — exact match or wildcard "*".
