Create `src/composer/layout.py` with:

`auto_layout(graph: WorkflowGraph, x_spacing: float = 400, y_spacing: float = 150, start_x: float = 100, start_y: float = 200) -> None`:

Algorithm (DAG layer assignment, left-to-right flow):
1. Build adjacency from graph's links: for each link, source_node -> target_node.
2. Find root nodes (no incoming connections).
3. Assign layers using longest path from any root: BFS/DFS from roots, each node gets layer = max(predecessors' layers) + 1. Roots get layer 0. Handle cycles by breaking them (skip back-edges).
4. Group nodes by layer.
5. Within each layer, sort nodes by their original vertical position or insertion order.
6. Assign positions: node.pos = [start_x + layer * x_spacing, start_y + row_in_layer * y_spacing].
7. Mutate graph nodes' pos in place.

Also add a helper `get_node(node_id: int) -> GraphNode` and `get_nodes() -> list[GraphNode]` to WorkflowGraph if not already present (needed for layout to access nodes).

Add tests:
- Test: auto_layout assigns distinct positions (no overlapping nodes)
- Test: auto_layout places source nodes left of their targets (lower x for earlier layers)
- Test: auto_layout handles single-node graph without error
- Test: auto_layout handles disconnected nodes (no links) — all placed in layer 0 with vertical spacing
