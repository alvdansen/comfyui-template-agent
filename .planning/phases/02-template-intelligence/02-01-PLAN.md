  <action>
    **Create `src/templates/cross_ref.py`:**

    - Import TemplateCategory, TemplateSummary from models, fetch_template_index, fetch_workflow_json, extract_node_types from fetch
    - Module-level `_cache = DiskCache()` singleton

    - `build_pack_index(categories: list[TemplateCategory]) -> dict[str, list[TemplateSummary]]`:
      - Iterate all templates across all categories
      - For each template, iterate `template.requiresCustomNodes` (list of pack IDs)
      - Build `pack_id -> [TemplateSummary(name, title, mediaType, tags, category=cat.category)]` mapping
      - Return the dict

    - `build_node_index(categories: list[TemplateCategory]) -> dict[str, list[TemplateSummary]]`:
      - For each template, call `fetch_workflow_json(template.name)` to get workflow
      - If workflow fetched, call `extract_node_types(workflow)` to get class_types
      - Build `class_type -> [TemplateSummary]` mapping
      - Cache the full built index with key "template_node_index" and templates TTL
      - Return the dict

    - `cross_reference(query: str, level: str = "pack", force_refresh: bool = False) -> dict`:
      - `level` is "pack" or "node"
      - Fetch template index (force_refresh)
      - If level="pack": build pack index, look up query in pack index
      - If level="node": check cache for node index first, build if miss, look up query
      - Optional fuzzy matching: if exact match returns nothing, try substring match on keys (e.g., "KSampler" matches "KSampler", "KSamplerAdvanced", "KSamplerSelect")
      - Return dict with:
        - `query`: the search query
        - `level`: "pack" or "node"
        - `exact_matches`: list of TemplateSummary dicts for exact matches
        - `fuzzy_matches`: list of TemplateSummary dicts for fuzzy matches (only if exact is empty)
        - `total_count`: total number of templates matched
        - `top_examples`: first 3 matches with name, title, and category (per user decision: "count + examples, top 3")

    - `format_cross_reference(result: dict) -> str`:
      - Format as human-readable string
      - "Used in {count} templates" header
      - Show top 3 example templates with name, category
      - If fuzzy matches, label as "Related matches (fuzzy)"

    - Add `if __name__ == "__main__"` with argparse: `--query`, `--level pack|node`, `--refresh`
  </action>
