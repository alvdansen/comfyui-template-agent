  <action>
    **Create `src/templates/coverage.py`:**

    Imports: fetch_template_index, flatten_templates from fetch; build_pack_index from cross_ref; NodePack from registry.models; classify_node from shared.categories; DiskCache from shared.cache; math, collections.Counter from stdlib.

    **Gap Analysis (TMPL-04):**

    - `score_gap_opportunity(node_pack: NodePack, template_count: int) -> float`:
      - If template_count > 0: return 0.0 (not a gap)
      - dl_score = math.log(node_pack.downloads + 1, 10) (log10 scale)
      - star_score = math.log(node_pack.github_stars + 1, 2) (log2 scale)
      - return dl_score * (1 + star_score * 0.5)

    - `suggest_template_idea(node_pack: NodePack) -> str`:
      - Based on classify_node(node_pack.name, node_pack.description) categories:
        - "video": "Could pair with img2vid or video effects workflow"
        - "image": "Could pair with txt2img or image processing workflow"
        - "audio": "Could pair with text-to-speech or audio processing workflow"
        - "3d": "Could pair with 3D generation or depth-based workflow"
        - default: "Could serve as utility node in multi-step pipeline"
      - Include the node description truncated to 100 chars for context

    - `gap_analysis(by_category: bool = False, limit: int = 20, force_refresh: bool = False) -> dict`:
      - Fetch all registry nodes via fetch_all_nodes(sort="downloads", limit=300)
      - Fetch template index via fetch_template_index(force_refresh)
      - Build pack index from template data
      - For each registry node pack:
        - Count templates using this pack (from pack index)
        - Score the gap opportunity
        - If score > 0, add to gaps list with: pack info, score, suggestion from suggest_template_idea
      - Sort gaps by score descending
      - If by_category: group gaps by classify_node category
      - Return dict with:
        - `total_gaps`: number of packs with no template
        - `total_packs_checked`: number of registry packs checked
        - `gaps`: top N gaps (list of dicts with name, id, downloads, stars, score, suggestion, categories)
        - `by_category` (if flag set): dict of category -> gaps list

    - `format_gap_analysis(result: dict) -> str`:
      - Default view: ranked list sorted by opportunity score
        - Each entry: "#{rank} {name} (downloads: {dl}, stars: {stars}) -- Score: {score:.1f}"
        - Below each: "  Suggestion: {suggestion}"
      - If by_category: grouped sections with category headers
      - Header: "Gap Analysis: {total_gaps} packs with no template coverage (checked {total_packs_checked})"

    **Coverage Report (TMPL-05):**

    - `coverage_report(force_refresh: bool = False) -> dict`:
      - Fetch template index
      - Flatten all templates
      - **Metric 1 - Templates per category**: Count templates by mediaType using Counter
      - **Metric 2 - Node coverage %**: Count unique packs in requiresCustomNodes across all templates, divide by total registry packs (from fetch_all_nodes). Pack-level metric.
      - **Metric 3 - Thin spots**: Categories with fewer than average templates per category
      - **Metric 4 - Growth trends**: Group templates by month using date field (YYYY-MM extraction), show templates added per month for recent 6 months
      - Return dict with: total_templates, templates_by_category, coverage_pct, thin_spots, growth_by_month

    - `format_coverage_report(result: dict) -> str`:
      - Summary header: "Template Library: {total} templates, {coverage_pct:.1f}% pack coverage"
      - Category table: category | count | bar chart (ASCII)
      - Thin spots section: list categories below average
      - Growth trends: month | templates added

    - Add `if __name__ == "__main__"` with argparse:
      - Subcommands: `gap` (--by-category, --limit), `coverage` (--refresh)
  </action>
